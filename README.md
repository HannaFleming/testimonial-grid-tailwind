## Testimonial Grid

Install dependencies

```
npm install
```
```
npm run watch
```
Install the plugin from npm:


```
npm install -D @tailwindcss/line-clamp
```
Then add the plugin to your tailwind.config.js file:
```
// tailwind.config.js
module.exports = {
  theme: {
    // ...
  },
  plugins: [
    require('@tailwindcss/line-clamp'),
    // ...
  ],
}
```
